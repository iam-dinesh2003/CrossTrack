### CrossTrack — AI-Powered Job Application Tracker
> **Java 17 · Spring Boot 3.2 · React 19 · MySQL · Docker · Gemini AI · Gmail API · Chrome Extension**

A **production-deployed** full-stack career platform I built solo from scratch.

**What it does:**
- Chrome Extension auto-captures applications from LinkedIn, Indeed & Handshake in under 2 seconds
- Gmail OAuth 2.0 auto-detects interview invites and rejection emails
- 6 AI features: Career Coach, Resume Match Score, Mock Interview, Interview Prep, Follow-up Generator, Application Autopsy
- Ghost job detection with automated 3-tier alert system (28 / 60 / 120 days)
- Kanban board + analytics dashboard + real-time WebSocket notifications

**By the numbers:**
- **30+** REST endpoints · **28** React components · **12** service classes
- **JWT** auth with refresh tokens · **Spring Scheduling** for automated alerts
- Deployed: Backend on **Railway** (Docker) · Frontend on **Vercel**
